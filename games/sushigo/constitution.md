# Constitution

- This game (as any other game inside this platform) is to be developed as a CAP plugin
- Use test driven development
- Use most modern JS features
- Use the SAP CAP MCP server to query information about best practices and documentation
- This is a card game. The business logic of each card is to be developed using the strategy pattern (because most cards behave the same but some require extra logic)
- This is a multiplayer game to be played in the browser. The technology of choice to enable realtime gaming is websockets
